# Portfolio Personal - Guillermo Sánchez

Repositorio del portafolio profesional de Guillermo Sánchez. Este proyecto ha sido desarrollado como una demostración técnica de capacidades frontend, utilizando un stack moderno enfocado en rendimiento, arquitectura de componentes y diseño responsivo.

## Descripción General

Esta aplicación web actúa como punto central para documentar y presentar mi experiencia como Desarrollador de Aplicaciones Multiplataforma. A diferencia de un currículum tradicional, este proyecto permite visualizar la implementación real de código, integraciones y decisiones de diseño.

El objetivo ha sido crear una interfaz limpia y eficiente que resalte los tres pilares de mi perfil:
1. Desarrollo Full Stack (Spring Boot / Angular).
2. Integración Hardware/Software (IoT con Arduino).
3. Automatización de sistemas (Python Scripting).

## Stack Tecnológico

El proyecto está construido sobre un entorno de desarrollo moderno para garantizar la optimización de recursos y velocidad de carga:

- **Framework:** React 18
- **Estilos:** Tailwind CSS (Diseño responsive y sistema de utilidad)
- **Empaquetador (Bundler):** Vite
- **Iconografía:** Lucide React
- **Control de Versiones:** Git / GitHub

## Estructura del Proyecto

El código fuente sigue una estructura modular para facilitar el mantenimiento y la escalabilidad:

/src
  /assets       # Recursos estáticos
  App.jsx       # Componente principal (Landing Page)
  main.jsx      # Punto de entrada de la aplicación
  index.css     # Configuración global de estilos y directivas de Tailwind
/public         # Archivos públicos estáticos

## Instrucciones de Ejecución

Para desplegar este proyecto en un entorno local, siga los siguientes pasos:

1. Clonar el repositorio:
   git clone https://github.com/grilletee/portfolio-2026.git

2. Acceder al directorio del proyecto:
   cd portfolio-2026

3. Instalar las dependencias necesarias:
   npm install

4. Iniciar el servidor de desarrollo:
   npm run dev

La aplicación estará disponible por defecto en http://localhost:5173.
